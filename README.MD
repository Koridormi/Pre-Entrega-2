# PreEntrega JavaSript 2

## Ubicacion del codigo JS

* src -> js -> app.js

## Se utilizo Gulp para procesar SASS y JS

* Los Archivos se procesan y crean al ejecutar el comando npm run dev.
* Al realizar esto, el proyecto funciona correctamente, sin la necesidad de tocar el codigo.

## Requisitos

* Algun Editor de Codigo, recomendacion: VS Code.
* Tener NodeJS y NPM instalados en tu sistema.
